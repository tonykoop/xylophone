# Xylophone Warm-Up: F Major Scale Walk

```
X:1
T:F Major Scale Warm-Up
M:4/4
L:1/8
K:F
f2 g2 a2 b2 | c'2 d'2 e'2 f'2 | f'2 e'2 d'2 c'2 | b2 a2 g2 f2 |
```
(In K:F, b = Bb automatically)

## Pentatonic warm-up (F pentatonic: F G A C D)
```
X:2
T:F Pentatonic Warm-Up
M:4/4
L:1/8
K:F
f2 g2 a2 c'2 | d'2 c'2 a2 g2 | f2 a2 c'2 d'2 | c'2 a2 g2 f2 |
```

**Practice tip:** The xylophone decays faster than glockenspiel — focus on clean attacks
rather than sustained tone. Each note is a fresh stroke.
